# Game State Schema

## Feature Description

Define the JSON schema for `state.json` matching the architecture spec: turn number, activePlayer, dice, diceUsed, board array, bar, home, lastMove, status, updatedAt.

## Context

This is roadmap item #1 for GitGammon - a backgammon game played on GitHub Pages through commits.

## Raw Requirements

- Define JSON schema for game state
- Include turn number tracking
- Track active player
- Represent dice state (rolled and used)
- Model board array structure
- Track pieces on the bar
- Track pieces in home
- Record last move
- Game status field
- Timestamp for updates (updatedAt)
- Must match the architecture specification
